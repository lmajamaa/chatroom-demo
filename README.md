# chatroom-demo

Chatroom demo application using following frameworks:
 * Node.js + Socket.io
 * React + TypeScript
 * Redis
 * material-ui

To run:
 * git clone https://github.com/lmajamaa/chatroom-demo.git
 * cd chatroom-demo
 * npm install
 * npm start
 * http://localhost:3000
